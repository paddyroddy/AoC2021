## Execute solutions

For each day there is a folder i.e. `day1` with a file `day1.py` in it.
## Testing

In top directory run `pytest -v`.
## Formatting

Execute `pip install -r requirements.txt; pre-commit install` to install git hooks in your `.git/` directory.
